# Documentación de Base de Datos

El modelo de datos se define con Prisma en
`packages/database/prisma/schema.prisma` e incluye `AuditLog` desde el inicio.

Entidades principales: User, ClientProfile, ProviderProfile, Category,
ServicePackage, Task, Offer, Booking, Payment, LedgerAccount, LedgerEntry,
Payout, Review, Dispute, ChatThread, ChatMessage, ModerationFlag, MediaFile,
Notification, AuditLog.
